# FastAPI Template

## Prepare

```bash
pip install -r requirements.txt
```

## Add Routes

1. Create new python file in `views`
2. Define the `APIRoute` object
3. Import the object in `views/__init__.py`

## Run

```bash
uvicorn main:app --reload
```


## Lint

```bash
make lint
```

## Test

```bash
make test
```
